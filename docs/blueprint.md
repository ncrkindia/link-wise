# **App Name**: LinkWise

## Core Features:

- Link Shortening: Generate shortened links without login.
- User Authentication: Authenticate users using Keycloak to manage links.
- Link Management: Logged-in users can manage their previously shortened links using email from JWT token.
- Advanced Options: Generate short links with optional password protection, expiration date, and QR code generation.
- Admin Dashboard: Admin dashboard with statistical analysis to mark users as blocked and delete links.

## Style Guidelines:

- Primary color: Soft blue (#90AFC5) to convey trust and calmness.
- Background color: Light gray (#F0F4F8) for a clean and modern feel (light scheme).
- Accent color: Warm orange (#D9534F) for key interactive elements and alerts.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use a consistent set of simple, line-based icons to represent actions and link attributes.
- Employ a responsive, grid-based layout for optimal viewing on all devices.
- Subtle transitions and animations on button clicks and form submissions to enhance interactivity.
